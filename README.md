## UTILIZAÇÃO

### LOCAL

CLONE O REPOSITÓRIO:

```bash
git clone https://github.com/digitalbocca/edb-node-pdf.git
```

INSTALE AS DEPENDÊNCIAS:

```bash
npm install
```

EXECUTE O ARQUIVO:

```bash
npm run dev
```
